Notes
-----

Installation sans réseau ? Oui !

Inclure elvis.


Groupe de paquets A
-------------------

  * a/aaa_base : arborescence du système

  * a/aaa_elflibs : collection de bibliothèques de base requises par beaucoup
    de logiciels

  * a/aaa_terminfo : descriptions de terminaux, requises pour en utiliser un

  * a/acl : outils d'administration des listes de contrôle d'accès

  * a/attr : outils de gestion des attributs étendus

  * a/bash : l'interpréteur de commandes Bourne Again Shell

  * a/bin : collection d'utilitaires communément utilisés par les scripts
    système

  * a/coreutils : collection de commandes de base, en particulier toutes
    celles qui ont trait à la gestion des fichiers et répertoires

  * a/cpio : format d'archivage permettant de créer des paquets RPM, également
    requis pour pouvoir créer l'initrd nécessaire à l'emploi d'un noyau
    générique

  * a/cxxlibs : bibliothèques assurant la compatibilité des applications liées
    à de plus anciennes versions de la bibliothèque standard C++ avec celle
    utilisée par le système

  * a/devs : peuple le répertoire '/dev', qui contient les points d'accès
    entre le système et le matériel

  * a/dialog : utilitaire permettant de créer des interfaces graphiques en
    mode texte à partir de scripts shell, utilisé par la plupart des outils
    Slackware

  * a/e2fsprogs : outils de gestion des systèmes de fichiers ext2, ext3 et
    ext4. 

  * a/etc : fichiers de configuration du système

  * a/file : contient un outil pour déterminer le type d'un fichier ou des
    fichiers donnés

  * a/findutils : programmes de recherche de fichiers

  * a/gettext : outils pour l'internationalisation et la localisation

  * a/glibc-solibs : bibliothèques C partagées du système, contenant les
    routines essentielles à son fonctionnement

  * a/glibc-zoneinfo : permet de configurer le fuseau horaire de la machine

  * a/grep : programme de recherche de chaînes de caractères à l'intérieur de
    fichiers

  * a/gzip : programmes de compression et décompression de fichiers

  * a/kbd : contient les fichiers de tables de caractères, les polices de la
    console et des outils pour le clavier

  * a/kernel-firmware : firmwares permettant de gérer certains matériels et
    distribués séparément du reste du noyau

  * a/kernel-generic-* : un noyau entièrement modulaire, qui n'embarque aucun
    support de système de fichiers et doit donc être utilisé avec un initrd

  * a/kernel-huge-* : le noyau par défaut de l'installation, avec le support
    des systèmes de fichiers

  * a/kernel-modules-* : les modules des noyaux

  * a/kmod : contient des bibliothèques et des outils pour charger des modules
    du noyau

  * a/less : visualiseur de fichiers texte

  * a/lilo : chargeur de démarrage

  * a/lvm2 : gestionnaire de volumes logiques

  * a/mdadm : outils d'administration pour le RAID logiciel

  * a/mkinitrd : utilitaire nécessaire pour créer l'initrd requis par le noyau
    générique

  * a/openssl-solibs : bibliothèques partagées de cryptographie

  * a/patch : programme permettant de modifier et de créer des fichiers en
    appliquant un fichier correctif

  * a/pciutils : utilitaires de configuration du matériel PCI

  * a/pkgtool : outil de gestion des paquets Slackware

  * a/procps : contient des programmes pour afficher des informations sur les
    processus en cours d'exécution

  * a/sed : éditeur de flux

  * a/shadow : stocke les mots de passe des utilisateurs sous forme cryptée

  * a/sysklogd : programmes pour les messages de traces système

  * a/sysvinit* : contrôle du démarrage, de l'exécution et de l'arrêt du
    système

  * a/tar : programme d'archivage

  * a/udev : programmes pour la création dynamique de fichiers de périphérique

  * a/util-linux : collection d'utilitaires nécessaires au fonctionnement d'un
    système Linux

  * a/which : affiche le chemin complet d'une commande installée dans le PATH

  * a/xz : programmes de compression et de décompression de fichiers


Groupe de paquets AP
--------------------

  * ap/bc : langage de traitement des nombres à précision arbitraire

  * ap/diffutils : programmes montrant les différences entre fichiers ou
    répertoires

  * ap/gawk : programmes de manipulation de fichiers texte

  * ap/groff : programmes de formatage de texte, utilisés principalement pour
    les pages man

  * ap/man : programmes pour rechercher et afficher des pages man

  * ap/man-pages : les pages man du Linux Documentation Project (LDP) en
    version originale

  * ap/slackpkg : outil pour l'installation ou la mise à niveau des paquets
    Slackware à travers un réseau

  * ap/texinfo : programmes de lecture, écriture et conversion des pages info

  * ap/vim : l'éditeur de texte Vim


Groupe de paquets D
-------------------

  * d/autoconf : programmes produisant des scripts shell qui configurent
    automatiquement le code source

  * d/automake : générateur de Makefile 

  * d/bison : analyseur syntaxique

  * d/flex : analyseur lexical

  * d/gcc : contient la partie de la collection de compilateurs GNU utilisée
    pour compiler le code source C 

  * d/gcc-g++ : support C++ pour la collection de compilateurs GNU

  * d/git : système de contrôle de versions

  * d/gperf : générateur de fonction de hachage

  * d/gnupg : outil GNU pour effectuer de la communication et du stockage
    sécurisés

  * d/intltool : outil d'internationalisation utilisé pour extraire des
    chaînes traduisibles à partir de fichiers sources

  * d/kernel-headers : fichiers d'en-tête C du noyau Linux 

  * d/libtool : script de support de bibliothèques GNU génériques, qui emballe
    la complexité d'utilisation de bibliothèques partagées dans une interface
    cohérente et portable

  * d/make : programme pour compiler des paquets

  * d/m4 : processeur de macros

  * d/perl : langage pratique d'extraction et de rapport

  * d/pkg-config : outil utilisé lors de la compilation d'applications et de
    bibliothèques. 

  * d/python : l'environnement de développement Python


Groupe de paquets L
-------------------

  * l/expat : bibliothèque C orientée flux pour analyser de l'XML

  * l/gdbm : contient le GNU Database Manager, une bibliothèque de fonctions
    de base de données

  * l/glibc : bibliothèque C standard

  * l/glibc-i18n : traductions pour la bibliothèque C standard

  * l/glibc-profile : versions statiques des bibliothèques C 

  * l/gmp : contient des bibliothèques de maths

  * l/libcap : implémente les interfaces niveau utilisateur avec les fonctions
    qui établissent le partage avec les privilèges root dans un ensemble de
    droits distincts

  * l/libmpc : contient une bibliothèque pour le calcul arithmétique de
    nombres complexes avec une haute précision au choix et l'arrondissement
    correcte du résultat

  * l/mpfr : contient des fonctions pour des maths à précision multiple

  * l/ncurses : bibliothèque qui fournit des fonctions de définition de
    touches du clavier et de couleurs d'écran

  * l/readline : ensemble de bibliothèques qui offrent des fonctionnalités
    d'édition et d'historique pour la ligne de commande

  * l/zlib : contient des routines de compression et décompression utilisées
    par quelques programmes


Groupe de paquets N
-------------------

  * n/ca-certificates : liste de certificats pour les connexions sécurisées

  * n/curl : contient un utilitaire et une bibliothèque utilisés pour le
    transfert de fichiers

  * n/cyrus-sasl : simple couche d'authentification et de sécurité, une
    méthode pour ajouter le support d'authentification aux protocoles basés
    sur la connexion

  * n/dhcpcd : client DHCP

  * n/iproute2 : outils réseau basiques et avancés

  * n/iputils : outils réseau que l'on trouve sur tous les systèmes *NIX

  * n/links : navigateur Web en mode texte

  * n/net-tools : outils réseau de base 

  * n/netkit-* : collection d'outils réseau de base

  * n/network-scripts : scripts et fichiers de configuration réseau de base

  * n/openssh : ensemble d'outils permettant des communications sécurisées en
    utilisant le protocole SSH

  * n/telnet : outil de communication non sécurisé

  * n/tftp-hpa : interface utilisateur pour le protocole simplifié de
    transfert de fichiers, utilisé notamment pour le démarrage réseau

  * n/traceroute : permet d'afficher la route réseau que les paquets prennent
    pour accéder à un hôte donné

  * n/wget : outil pour le téléchargement non-interactif de fichiers distants
