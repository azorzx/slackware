Migrate from Xfce or MATE to KDE
================================

This document explains how to upgrade an existing MLED installation based on
Xfce or MATE to the new KDE-based version without doing a fresh installation.

Switch to init 3.

  # telinit 3

Uncomment all third-party repositories in '/etc/slackpkg/slackpkgplus.conf'
and refresh information on available packages:

  # slackpkg update

Clean your system:

  # slackpkg upgrade-all
  # slackpkg clean-system

Remove all remaining MLED packages:

  # removepkg /var/log/packages/*microlinux*

Import the Microlinux helper scripts:

  # git clone https://github.com/kikinovak/slackware

If you already have these, then 'cd' into the file tree and update it:

  # cd slackware
  # git pull

Trim your base system:

  # cd MLED-14.1-$ARCH/tools/
  # ./trim_desktop_base.sh

This script will take care of two things:

  a. install required packages 
  b. remove unnecessary packages

Remove any remaining configuration of 'slackpkg+':

  # rm /etc/slackpkg/slackpkgplus.conf*

Download and install the custom 'slackpkg+' package provided by MLED:

  # links http://www.microlinux.fr/slackware/MLED-14.1-32bit/slackware/ap/

On 64-bit Slackware:

  # links http://www.microlinux.fr/slackware/MLED-14.1-64bit/slackware/ap/

Grab the package from the 'ap/' directory and install it manually.

  # installpkg slackpkg+-*

Update system information:

  # slackpkg update gpg
  # slackpkg update

Upgrade:

  # slackpkg upgrade-all

This will replace a handful of packages like 'kde-baseapps', 'MPlayer', etc.
It will also reinstate your Multilib packages if they got removed at an
earlier stage. 

On 64-bit Slackware, install the remaining Multilib packages:

  # slackpkg install compat32

Now install the MLED package collection:

  # slackpkg install microlinux

From now on, follow the official installation guide in the 'README.txt'. 

Since your user has already been created, the default user profile provided by
the 'user-settings-kde' package won't apply. You can copy it over manually.
Log into your account and do this:

  $ cd
  $ cp -Rv /etc/skel/.config .
  $ cp -Rv /etc/skel/.kde .

Switch to init 4:

  # init 4

Have fun!
